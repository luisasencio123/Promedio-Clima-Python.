En la programación tradicional el programa se hace usando funciones. En este trabajo primero se pidió las temperaturas de cada día y luego, en otra función, se sacó el promedio semanal. Este tipo de programación es más directa y fácil cuando el programa no es muy grande, ya que se entiende rápido lo que hace cada parte.

Por otro lado, en la programación orientada a objetos se trabajó con una clase llamada clima, donde se guardan las temperaturas y se usan métodos para hacer los cálculos. De esta forma el código queda más ordenado y es más fácil de usar si después se quiere agregar más cosas al programa.

Para finalizar, se puede decir que ambos tipos de programación funcionan bien, pero la programación tradicional es mejor para ejercicios simples, mientras que la programación orientada a objetos ayuda a tener un código más organizado cuando el programa es más grande.
